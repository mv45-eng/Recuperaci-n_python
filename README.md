Descripción: Bubble Sort es un algoritmo de ordenamiento bastante simple: va comparando números que están uno al lado del otro y, si están en el orden incorrecto, los intercambia. Hace esto varias veces hasta que todo queda ordenado. En cada pasada, el número más grande va subiendo hasta el final de la lista, como si fuera una burbuja

Complejidad Computacional

¿Cuál es la complejidad Big O? 

La complejidad es O(n²) porque el algoritmo utiliza dos bucles anidados que recorren la lista, generando aproximadamente n × n comparaciones.

¿Qué pasa con el tiempo de ejecución si la lista tiene 10 elementos vs 100 elementos?

Si la lista tiene 10 elementos, el número de operaciones será aproximadamente 100.  

Si la lista tiene 100 elementos, el número de operaciones será aproximadamente 10,000.

El tiempo de ejecución crece muy rápido con respecto al tamaño de la lista.
